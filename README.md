# Mobile Listing Auto-Updater (Python)

A Python-based **automation tool** designed to manage and update mobile phone listings on a web platform through browser interaction.

This project demonstrates practical experience in **web automation**, **DOM interaction**, and **time-based task execution** using Python.

---

## üìñ Project Overview

The purpose of this tool is to automate repetitive listing management tasks that are typically performed manually through a web interface.

The script:
- Logs into a user account
- Navigates through multiple pages of listings
- Selects listings one by one
- Performs update actions on each item
- Repeats the process at randomized time intervals

The goal is to keep listings active and up to date without continuous manual interaction.
